## üìã Quick Reference

### Key Concepts
- **PriceList**: Entity representing pricing tier (Retail, Wholesale, etc.)
- **ItemPrice**: Junction table linking Item ‚Üí PriceList ‚Üí Price
- **Option C**: User explicitly selects price list when creating invoice
- **Auto-population**: Prices automatically populated from selected price list

### Key Files Modified
- `invoice.service.ts` - DTO with priceListId
- `item-price.service.ts` - Method to fetch prices
- `invoice-form.component.ts` - All logic for price list selection
- `invoice-form.component.html` - Template with dropdowns

### Key API Endpoints
- `GET /api/pricelist` - Fetch all price lists
- `GET /api/itemprice/pricelist/{id}/items` - Fetch items with prices
- `POST /api/invoice/create` - Create invoice with priceListId
- `PUT /api/invoice/{id}` - Update invoice with priceListId

### Key User Actions
1. Open invoice form
2. Default price list auto-selected (e.g., "Retail")
3. Optionally change price list to "Wholesale"
4. Add line items - prices auto-populate
5. Submit invoice with selected pricing
